=======================================================================
                    CodeCheck AI - 代码与PRD智能质检工具
=======================================================================

【产品简介】
CodeCheck AI 是面向软件/互联网企业研发团队的代码与PRD需求一体化智能质检工具。
所有AI检测逻辑均在浏览器本地模拟运行，无需后端接口，数据不上传公域大模型。

【核心功能】
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 代码智能审查     │ 支持 JS/Python/Java/Go 代码片段自动审查       │
│                    │ 风险分级高亮（高危/中危/低危）                  │
│                    │ AI一键生成修复代码与修改理由                     │
├─────────────────────────────────────────────────────────────────────┤
│ 2. PRD需求质检      │ 自动扫描需求歧义、边界缺失、逻辑冲突            │
│                    │ 任务自动拆解（前端/后端/测试）                  │
│                    │ 自动生成功能测试用例                            │
├─────────────────────────────────────────────────────────────────────┤
│ 3. 批量数据报表     │ 团队缺陷统计饼图、月度质检数据表格              │
│                    │ PDF导出模拟（技术总监专属）                      │
├─────────────────────────────────────────────────────────────────────┤
│ 4. 角色权限系统     │ 实习生 / 普通开发 / 技术总监                   │
│                    │ 细粒度权限隔离，适配研发组织架构                  │
└─────────────────────────────────────────────────────────────────────┘

【快速开始】
1. 解压本压缩包到任意文件夹
2. 直接双击打开 index.html（推荐使用 Chrome/Edge 浏览器）
3. 点击页面内"加载演示素材"按钮，快速体验AI质检效果
4. 通过右上角下拉框切换不同角色，体验权限差异

【页面导航】
┌─────────────────────────────────────────────────────────────────────┐
│ index.html       → 工作台首页（系统简介、快捷入口）                  │
│ code-review.html → 代码智能审查页（代码输入、风险检测、AI优化）      │
│ prd-check.html   → PRD需求质检页（文档输入、缺陷扫描、任务拆解）     │
│ reports.html     → 批量数据报表页（统计图表、月度数据、PDF导出）     │
└─────────────────────────────────────────────────────────────────────┘

【推荐体验流程】
1. 打开工作台首页 → 点击"代码智能审查"入口
2. 点击"加载演示素材" → 点击"开始AI审查" → 查看风险高亮
3. 点击"一键生成" → 查看AI优化后的代码
4. 返回首页 → 点击"PRD需求质检"入口
5. 点击"加载演示素材" → 点击"开始AI质检" → 查看缺陷与任务拆解
6. 切换至"技术总监"角色 → 访问"批量数据报表"查看图表

【安全特性】
✓ 所有代码与PRD文档仅本地解析，不上传公域大模型
✓ 企业私有技术资产加密存储，防泄露
✓ 操作记录全链路可溯源，满足审计要求
✓ 细粒度角色权限隔离，适配研发组织架构

【技术栈】
HTML5 + TailwindCSS + 原生 JavaScript + Chart.js + Lucide Icons

【注意事项】
- 本工具为纯前端演示版，所有AI检测逻辑均为本地模拟
- 建议使用现代浏览器（Chrome 90+ / Edge 90+）以获得最佳体验
- 无需安装任何服务器，解压即可直接运行
